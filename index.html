The Model

Built with purpose

Specialized Agents, Not Generalists

Each agent has a narrow, deep focus — like hiring a domain expert rather than a generalist. The Legal SME knows TILA, FCRA, and GLBA cold. The Finance SME owns APR calculations and Reg Z compliance. No agent operates outside their domain.

Mandatory Gates, Not Optional Reviews

Six non-negotiable checkpoints — Architecture, Security, Legal, Finance, Credit Policy, and Release — that work cannot bypass under any schedule pressure. Each gate requires sign-off from the designated agent before the next phase begins.

Human-in-the-Loop Oversight

Every gate is a human decision point. The agents do the work — they research, specify, design, and implement. Humans approve the decisions. This isn't autonomous AI; it's AI-augmented development with deliberate human control at every critical juncture.

Built-in compliance controls

Architecture Gate

New service, data model change, or external integration. Work cannot proceed until approved.

Sign-off: Architect
Security Gate

Features handling PII, authentication, or financial transactions. STRIDE threat model required.

Sign-off: Cybersecurity SME
Legal Gate

Features touching disclosures, consumer data, credit decisions, or financial communications.

Sign-off: Legal SME
Finance Gate

Loan calculations, payment processing rules, product eligibility logic, or APR disclosures.

Sign-off: Finance SME
Credit Policy Gate

Credit eligibility thresholds, pricing tier structures, max finance amounts, or decisioning logic.

Sign-off: Credit & Risk SME
Release Gate

All production deployments. Dual sign-off required — both QA and Security must independently approve.

Sign-off: QA Analyst + Cybersecurity SME

Microservices
Prequal Application Service
Core application flow API. Manages server-side form state across all steps — personal info, vehicle selection, dealer selection, identity verification, and FCRA consent. Owns application lifecycle and confirmation number generation.
Identity & Auth Service
OTP generation (6-digit CSPRNG) and delivery via Amazon SNS. KBA orchestration with stubbed emulation for MVP. Session token lifecycle via Auth0 — rotation after OTP success, reuse detection, 60-min idle / 4-hour absolute timeout.
Credit Decisioning Engine
Soft-pull bureau orchestration, credit policy evaluation, and tier assignment (Preferred → Access). Max finance amount lookup, adverse action reason code generation, and FCRA disclosure gate enforcement before pull execution.
Lead Delivery Service
ADF payload construction and delivery to dealer DMS integrations — DealerTrack and RouteOne. Handles retry logic with exponential backoff, delivery confirmation, idempotency keys, and lead status tracking. PREQUALIFIED outcomes only.
Notification Service
Transactional email delivery for prequalification confirmations and adverse action notices (Path A — KBA failure, Path B — soft pull decline). FCRA-compliant timing enforcement. Coordinates OTP SMS delivery via Amazon SNS.
Dealer & Vehicle Catalog
Toyota and Lexus franchise dealer registry, new vehicle inventory, dealer enrollment state, and ADF routing configuration. Enforces new-vehicle-only scope. Provides dealer selection UI data and pre-population logic after vehicle changes.
